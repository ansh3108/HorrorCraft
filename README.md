## A horror shaderpack that transforms the game into a bodycam experience. 

### With features like volumetric mist, lens distortions and hallucinations. 
 
 ------
 
### Atmosphere

* Void Darkness
* Ground Mist
* Black water
* Corrupted sky
* Blood Moon

### Paranoia

* "Something Moved" Blur
* Shadow Figures
* Sanity Warp(the deeper you dig, the worse it gets)
* Cold Breath

### Optics

* Dynamic Exposure & Desaturation
* Failing Infrastructure
* Valley shadows
* Wet Lens
